# Transcend repositioning experience concept

This is a lightweight GitHub Pages concept site for a Product Marketing Manager take-home project.

The experience reframes Transcend's narrative around a stronger enterprise category story:

1. Automate privacy rights and consent
2. Govern customer data at the systems layer
3. Activate AI responsibly

Company stage remains a useful entry point, but the intended destination is enterprise infrastructure: customer data becomes a governed, AI-ready asset.

## Files

- `index.html` - Main page structure
- `style.css` - Visual design and responsive layout
- `script.js` - Stage switching, mega-menu interactions, and audio trigger logic
- `transcend-pulse.mp3` - Subtle enterprise confirmation sound
- `index-standalone.html` - Optional all-in-one version with CSS and JavaScript embedded

## Audio Behavior

The audio cue does not autoplay, does not loop, and does not show controls. It only plays after meaningful enterprise actions:

- Selecting the Enterprise stage
- Clicking Enterprise in the Solutions concept menu
- Clicking Explore platform when the enterprise state is active

## GitHub Pages

For the standard version, upload:

- `README.md`
- `index.html`
- `style.css`
- `script.js`
- `transcend-pulse.mp3`

For the all-in-one version, rename `index-standalone.html` to `index.html` before uploading, then also upload `transcend-pulse.mp3`.

This is a repositioning concept and is not official Transcend navigation or brand guidance.
